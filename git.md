### Git config
Various config stuff.
<br />
<br />
Changing editor for commits.
<br />
`git config --global core.editor "nvim"`
<br />
<br />
Add username and email.
<br />
`git config --global user.name "Your Name"`
<br />
`git config --global user.email "youremail@yourdomain.com"`
